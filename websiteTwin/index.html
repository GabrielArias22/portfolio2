Magic Gizzball

The Magic Gizzball is a cat themed magic 8 ball that uses an array to hold a list of random responses that are called when the button is hit. This was made in P5JS using Javascript and it's purpose is to help make decisions like a traditional magic 8 ball, however the Magic Gizball can hold as many custom responses as needed.
